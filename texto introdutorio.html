Este Website interativo foi desenvolvido com o objetivo de atender a uma demanda por conteúdo de nível avançado, para os estudantes de programação. As perguntas são sobre o front end (HTML5, CSS3 e JavaSprit),


já que são elas o primeiro contato que os alunos tem com a construção de um projeto utilizando linhas de código. Os cursos em instituições de ensino reconhecidos pelo MEC oferecem muita teoria essencial para o desenvolvimento do aluno,
mas muita pouca prática na resolução de problemas (linhas de código).Isso dificulta o início da carreira profissional de qualquer futuro programador.
